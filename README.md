# FocusFlow – ADHD-Friendly Daily Task Manager

OpaList is a JavaFX-based productivity application designed to help users organize daily tasks in a simple, 
low-overwhelm environment. The application focuses on category-based task management, daily focus organization, 
and persistent task tracking with an ADHD-friendly workflow.

The app was designed around the idea of reducing cognitive clutter by allowing users to focus on one category of tasks 
at a time while still maintaining the option to view all tasks in a centralized dashboard.

---

# Features

- JavaFX graphical user interface 
- Category-based task organization 
- Dynamic category filtering
- "View All Tasks" dashboard
- Daily focus workflow
- Persistent task storage using JSON
- Automatic task rollover between days
- Completed tasks are automatically removed the next day
- Visual task completion styling
- Completed tasks sorted to the bottom of the task list
- Task persistence after closing the application 

---

# Categories

Tasks can be organized into the following categories

- Work
- School
- Self-care
- Family

--- 

# ADHD-Friendly Design Goals

This project was designed with several ADHD-friendly productivity principles in mind:

- Reduced visual clutter
- Focused category views
- Simple task interaction flow
- Persistent unfinished task rollover 
- Minimal-friction task management
- Clear visual distinction between completed and active tasks.
